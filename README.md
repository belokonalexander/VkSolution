# VkSolution
Разработать приложение, которое:

1. Авторизуется в VK.
2. Выкачавает ФИО и фотографию профиля.
3. Сохраняет информацию профиля и фотографию.
4. При повторном открытии приложения загружает фотографию и данные профиля на экран.
